# Release Notes

## 0.1.2

- Add an automatic picture-in-picture inset for distant locations, including swap and close interactions.
- Add custom photo-stack clusters for tightly grouped public art locations.
- Replace basic marker popups with richer desktop cards and mobile slide-up drawers that show image, artist, location, walking directions, type, and collection.
- Increase public map pins to 50px, remove the visible pin border, and raise hovered pins above nearby pins.
- Tune cluster positioning and radius so grouped locations stay aligned with their map coordinates.

## 0.1.1

- Add built-in GitHub update support so client sites can install future plugin updates from your public repository after one manual upgrade to this version.
- Add a configurable REST export feed for sharing published art locations, taxonomies, and image URLs with another site.
- Add plugin settings for enabling the export feed and securing it with an optional shared token.
- Add WordPress `readme.txt` metadata so update details display in the plugin modal.
